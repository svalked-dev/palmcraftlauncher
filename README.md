PalmLauncher

Привет! Это мой лаунчер для Minecraft под Linux. Я его написал, потому что мне надоело возиться с официальным лаунчером и всякими заморочками с авторизацией. Я просто хотел выбирать версию из списка и запускать игру, без лишней головной боли.
Как это работает

Я взял файл minecraft-server-jar-downloads.md, который лежит в репозитории. В нём таблица с версиями и ссылками на клиентские jar'ы. Лаунчер парсит этот файл, показывает все версии в настройках, а ты выбираешь нужную двойным кликом. Потом он скачивает jar и запускает майн с указанным количеством оперативки.

Для красоты я добавил размытый фон: если положить main.png рядом со скриптом, он будет отображаться с блюром по краям и чётким центром. Без картинки — просто тёмный фон.
Установка

Проще всего поставить зависимости через пакетный менеджер:
Arch
bash

sudo pacman -S python-pyqt6 python-requests python-pillow

Ubuntu / Debian
bash

sudo apt install python3-pyqt6 python3-requests python3-pil

Fedora
bash

sudo dnf install python3-pyqt6 python3-requests python3-pillow

Потом просто скачай репозиторий и запусти:
bash

git clone https://github.com/svalked-dev/palmcraftlauncher
cd palmcraftlauncher
python palmlauncher.py

Если хочешь использовать виртуальное окружение:
bash

python -m venv venv
source venv/bin/activate
pip install PyQt6 requests pillow
python palmlauncher.py

Костыли

Да, это не идеальный лаунчер. Я знаю, что некоторые версии могут не запускаться, потому что не все библиотеки подгружаются автоматически. Если вылезет ошибка NoClassDefFoundError, скорее всего нужно добавить недостающую библиотеку в список в коде. Я пока добавил основные (LWJGL, Brigadier, DataFixerUpper, Authlib), но для новых версий может не хватать чего-то ещё.

Ещё момент: выбор версии только в настройках, на главном экране её нет. Это сделано специально, чтобы не загромождать интерфейс. Просто открой настройки, найди версию в списке и кликни по ней дважды — она станет активной.
Настройки

Все настройки хранятся в ~/.palmlauncher/config.json. Там можно поменять версию и RAM вручную, если лень открывать GUI.

Никнейм сохраняется в ~/.palmlauncher/nick.txt.
